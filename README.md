# Neha Tripathi — Portfolio

A React portfolio with rich animations (Framer Motion), featuring freelancing hotel projects: **The Holy Awadh Hotel**, **Hotel The Balram**, and **Hotel Dhanraj Inn** (Ayodhya).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 18 + Vite
- Framer Motion (animations)
- React Router DOM
- CSS variables (DM Serif Display + Outfit fonts)

## Content

Edit `src/data/content.js` to update resume content, featured work, and contact details.
